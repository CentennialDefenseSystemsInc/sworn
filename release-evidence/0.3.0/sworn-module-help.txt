usage: sworn [-h] [--version]
             {init,check,report,status,verify,keygen,ci-check} ...

Deterministic, fail-closed AI code governance.

positional arguments:
  {init,check,report,status,verify,keygen,ci-check}
    init                Initialize sworn in a git repo
    check               Run gate pipeline on staged files
    report              Generate evidence report
    status              Show sworn status
    verify              Verify evidence chain integrity
    keygen              Generate Ed25519 signing keypair
    ci-check            Run gate pipeline on PR diff files

options:
  -h, --help            show this help message and exit
  --version             show program's version number and exit
